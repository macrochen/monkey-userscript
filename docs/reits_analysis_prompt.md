### 第二部分：⚠️ 致命红线检查 (Red Flags Check)
*请优先扫描文档，若发现以下情况直接报警，若无则标注“未发现异常”：*
1.  **特许经营权剩余年限**：若为特许经营权类，剩余年限是否 < 10年？
2.  **可供分配金额 (CAD) 达成率**：实际金额是否低于招募说明书/年度预测值的 90%？
3.  **核心运营数据暴跌**：出租率环比下降 > 3%？或 车流量/收费额同比下降 > 5%？
4.  **关联交易风险**：前三大租户中是否包含原始权益人的关联方？（是否存在“左手倒右手”做高业绩的嫌疑？）
5.  **费用异动**：管理费、托管费或运营成本是否同比大幅上升？

### 第三部分：关键数据透视 (Data Deep Dive)

*(请根据识别到的资产类型，选择对应维度的指标列出)*

**👉 如果是【产权类】 (园区/仓储/保租房/消费)：**
*   **出租率 (Occupancy Rate)**：期末出租率是多少？环比是升是降？
*   **租金单价 (Rent)**：平均有效租金是多少？是否存在“以价换量”（出租率稳但租金跌）的情况？
*   **租户结构 (WALE & Concentration)**：加权平均租赁到期日 (WALE) 是多少年？前十大租户占比多少？（占比过高有退租风险）
*   **资本化率 (Cap Rate)**：(若为招股书或年报) 评估采用的折现率/资本化率是多少？(越低估值越贵)

**👉 如果是【特许经营权类】 (高速/能源/环保)：**
*   **业务量**：日均车流量 / 上网电量 / 垃圾处理量。同比变化百分比？
*   **单价**：收费标准是否调整？(如高速过路费、电价国补退坡风险)
*   **营收转化**：营业收入的变动幅度是否与业务量变动匹配？

### 第四部分：现金流与分红能力 (The Cash)
*   **本期可供分配金额 (CAD)**：具体数字。
*   **年化分派率推算**：基于本期数据，推算的年化分派率约为多少？（简单计算：本期CAD * 4 / 当前市值，若无法获取市值则忽略）
*   **EBITDA**：(若是年报) 息税折旧摊销前利润走势。
